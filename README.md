# Product Game Audio Assets

Audio files for The Product Game - a multiplication strategy game for middle school students.

## Music
- **music-player1-mozart.mp3** - Mozart's *Eine kleine Nachtmusik* (Player 1's theme)
- **music-player2-vivaldi.mp3** - Vivaldi's *Spring* from Four Seasons (Player 2's theme)
- **music-rapid-rossini.mp3** - Rossini's *William Tell Overture* finale (Robo vs Robo rapid play)

## Sound Effects
- sfx-claim.mp3 - Successful product claim
- sfx-miss.mp3 - Product not on board
- sfx-blocked.mp3 - Product already claimed (lose turn)
- sfx-capture.mp3 - Capture opponent's product
- sfx-win.mp3 - Victory fanfare
- sfx-draw.mp3 - Stalemate/draw

All music is public domain classical recordings.
